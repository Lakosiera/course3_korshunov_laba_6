# Что это

## Django

```bash
"server"    # django проект (команда: django-admin startproject server)
│
├── "laba_6" # django модуль (команда: python manage.py startapp laba_6)
│   │
│   ├── "templates" # папка шаблонов для рендера вьюшек
│   │   └── index.html # шаблон корневой страницы
│   │
│   ├── "static" # папка со статическими ресурсами (стили, картинки и тд.)
│   │   └── ...
│   │
│   ├── admin.py    # админка моделей из 'models.py'
│   ├── apps.py     # класс для подключения в основной проект
│   ├── models.py   # модели данныйх для работы с базой и проектом
│   ├── urls.py     # настрока путей для модуля
│   └── views.py    # вьюшки страниц
│
└── "server" # django сервер
    │
    ├── settings.py # настроки django (подключенные модули, настрока подключения к базе)
    └── urls.py     # настрока путей проекта
```
